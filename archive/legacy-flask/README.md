# Legacy Flask Prototype

This folder contains the earlier Flask + SQLite prototype that previously lived at the repository root.

It is archived for reference only and is not the current source of truth for product scope, routing, contracts, or database design.

## Contents

- `app.py`: archived Flask entrypoint
- `asiamath.db`: archived SQLite database file
- `schema.sql`: archived SQLite schema
- `templates/index.html`: archived landing page template

## Running the Archived Prototype

Run it from inside this directory so its local SQLite database stays contained here:

```bash
cd archive/legacy-flask
python app.py
```
